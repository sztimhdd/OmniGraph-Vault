### 2.4 Cookie/Token Refresh Procedure

When `ret=200003 invalid session` appears (every ~14-31 days):

1. **DO NOT create new tabs** via `PUT /json/new` — triggers WeChat re-auth redirect
2. Open existing logged-in tab at `https://mp.weixin.qq.com/cgi-bin/home?t=home/index&lang=zh_CN&token=<NEW_TOKEN>`
3. Extract token from page URL
4. Extract cookies via CDP: `Network.getCookies` on mp.weixin.qq.com
5. Update `/root/OmniGraph-Vault/kol_config.py` (TOKEN + COOKIE, keep FAKEIDS intact)
6. `systemctl restart omnigraph-kol-scan-batch@{1..4}`
