# kb/static — Brand Assets & Static Files

This directory holds the static brand assets and (later) CSS/JS bundles for the
VitaClaw KB v2 SSG site. Files here are referenced by Jinja2 templates under
`kb/templates/` and served either by Caddy directly (production) or by
`python -m http.server` (local preview).

## Asset provenance (as of plan kb-1-04b, 2026-05-12)

| File | Status | Source |
|---|---|---|
| `favicon.svg` | **placeholder** | Generated by plan kb-1-04b — minimal "VC" mark on `#0f172a` background, matches dark-theme design tokens (UI-01) |
| `VitaClaw-Logo-v0.png` | **MISSING (stub)** | Real asset lives in vitaclaw-site sibling repo at `public/VitaClaw-Logo-v0.png`; not present locally at plan execution time. See `VitaClaw-Logo-v0.png.MISSING.txt` for graceful-degradation notes. |

Both files are referenced by `kb/templates/base.html` (delivered later by plan
kb-1-07) — `favicon.svg` via `<link rel="icon" type="image/svg+xml">`, and the
PNG via `<img src="/static/VitaClaw-Logo-v0.png" onerror="this.style.display='none'">`
so a missing PNG degrades cleanly to "no logo" without a broken-image icon.
