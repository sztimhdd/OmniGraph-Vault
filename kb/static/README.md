# kb/static — Brand Assets & Static Files

This directory holds the static brand assets and (later) CSS/JS bundles for the
VitaClaw KB v2 SSG site. Files here are referenced by Jinja2 templates under
`kb/templates/` and served either by Caddy directly (production) or by
`python -m http.server` (local preview).

## Asset provenance

| File | Status | Source | Sourced |
|---|---|---|---|
| `favicon.svg` | **placeholder** | Generated by plan kb-1-04b — minimal "VC" mark on `#0f172a` background, matches dark-theme design tokens (UI-01) | 2026-05-12 (kb-1-04b) |
| `VitaClaw-Logo-v0.png` | **SOURCED** (2048×2048 RGBA PNG, 480 KiB) | Copied from vitaclaw-site sibling repo (`public/VitaClaw-Logo-v0.png`) per V-3 | 2026-05-15 (out-of-band, ahead of kb-4 deploy) |

Both files are referenced by `kb/templates/base.html` — `favicon.svg` via
`<link rel="icon" type="image/svg+xml">`, and the PNG via
`<img src="/static/VitaClaw-Logo-v0.png" onerror="this.style.display='none'">`
so a missing PNG degrades cleanly to "no logo" without a broken-image icon.

The PNG was sourced out-of-band on 2026-05-15 ahead of the v2.2-7 YOLO deploy.
The corresponding `.MISSING.txt` placeholder stub was removed during kb-4-lite
plan-03 closure (2026-05-21) — see `.planning/phases/kb-4-ubuntu-deploy-cron-smoke/kb-4-03-SUMMARY.md`.
