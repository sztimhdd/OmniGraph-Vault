### Task 3: F2 — tmux helper script (commit 3)

**Files:**
- `scripts/cron_daily_ingest.sh` (new, executable bit set)
- `tests/unit/test_cron_daily_ingest.py` (new)

**Action:**
1. **New script** `scripts/cron_daily_ingest.sh`:
   - Shebang: `#!/usr/bin/env bash`
   - `set -euo pipefail`
   - Usage: `./scripts/cron_daily_ingest.sh [MAX_ARTICLES]` — `MAX_ARTICLES` defaults to `10`.
   - Compute `SESSION_NAME="daily-ingest-$(date +%Y%m%d)"` and `LOG_FILE="/tmp/daily-ingest-$(date +%Y%m%d-%H%M).log"`.
   - **Step A — same-day session detection:** `if tmux has-session -t "$SESSION_NAME" 2>/dev/null; then ...`. Inside: check pane is alive (`tmux list-panes -t "$SESSION_NAME"` exit 0 means alive). If alive → `echo "ERROR: session $SESSION_NAME already running" >&2; exit 1`. If dead pane → `tmux kill-session -t "$SESSION_NAME" || true; echo "Killed stale dead pane"`.
   - **Step B — cross-day stale sessions:** loop over `tmux list-sessions -F '#S' 2>/dev/null | grep '^daily-ingest-' || true` — for each not equal to today's `$SESSION_NAME`, kill it.
   - **Step C — launch tmux:** build a single chained shell command using **single-quoted heredoc-free** form to avoid escaping pain:
     ```bash
     CMD="cd $HOME/OmniGraph-Vault && \
       venv/bin/python scripts/cleanup_stuck_docs.py --all-failed && \
       PYTHONPATH=. /usr/bin/time -v venv/bin/python batch_ingest_from_spider.py \
         --from-db --max-articles ${MAX_ARTICLES} \
         2>&1 | tee ${LOG_FILE}"
     tmux new-session -d -s "$SESSION_NAME" "$CMD"
     ```
   - Use `${MAX_ARTICLES}` (parameter expansion already done) and `${LOG_FILE}` (host-evaluated). Avoid `$(...)` and `\$?` inside the tmux command body — chain with `&&` so failure short-circuits.
   - Final line: `echo "tmux session $SESSION_NAME launched, log: $LOG_FILE"`.
   - chmod +x via `git update-index --chmod=+x scripts/cron_daily_ingest.sh` after `git add` (executor must do this before commit).
2. **New test** `tests/unit/test_cron_daily_ingest.py` (Python subprocess-based; uses `pathlib`, `subprocess`, `shutil`):
   - `test_bash_n_passes` — `subprocess.run(["bash", "-n", "scripts/cron_daily_ingest.sh"])`, assert returncode==0.
   - `test_contains_cleanup_invocation` — read file content, assert `"cleanup_stuck_docs.py --all-failed"` in it.
   - `test_cleanup_before_ingest` — read file, find line numbers of `cleanup_stuck_docs.py` and `batch_ingest_from_spider.py`; assert cleanup line < ingest line.
   - `test_max_articles_default_10` — grep for `MAX_ARTICLES:-10` or `MAX_ARTICLES=${1:-10}` pattern; assert match.
   - `test_shellcheck_passes_or_skips` — `shutil.which("shellcheck")` is None → log warning, `pytest.skip("shellcheck not in PATH")`. Else `subprocess.run(["shellcheck", "scripts/cron_daily_ingest.sh"])`, assert returncode==0.
