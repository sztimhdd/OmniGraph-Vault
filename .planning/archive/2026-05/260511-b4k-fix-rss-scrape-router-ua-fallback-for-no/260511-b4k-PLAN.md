<objective>
Fix RSS scrape silent-empty bug in `lib/scraper.py:_scrape_generic`. Non-WeChat URLs (e.g. simonwillison.net link-blog posts) currently scrape empty because:

1. `_MIN_CONTENT_LENGTH=500` is too strict for short-form RSS posts (link blogs, brief notes typical of indie-dev RSS feeds).
2. Layer 2's single extractor (`trafilatura.extract(favor_precision=True)`) is too aggressive at filtering "boilerplate" on RSS-style sites — discards real content as noise.

Result: 45 RSS articles stuck in retry loop. User confirmed bare `requests.get(url, headers={'User-Agent': UA})` succeeds on all 45 — proving HTTP fetch works; the bug is in post-fetch extraction.

Purpose: Unstick the 45 articles by lowering the quality gate to 200 chars AND giving Layer 2 an extraction-fallback chain (precision → recall → html2text) so RSS-style content has a fighting chance.

- ~30-50 LOC change to `lib/scraper.py` (Layer 2 only — Layer 1 stays single-extractor fast path)
- ≥2 new pytest tests in `tests/unit/test_scraper.py`
- Smoke run on 3 stuck URLs via `scripts/local_e2e.sh wechat <url>` (note: harness name says wechat but `scrape_url` routes by URL host, so non-WeChat URLs exercise `_scrape_generic`)

    ```python
    def _extract_with_fallbacks(html: str) -> tuple[Optional[str], dict[str, int]]:
        """Layer 2 extraction-fallback chain. Try precision → recall → html2text;
        return (markdown, lengths) where markdown is the first passing extract or
        None if all three fail. lengths records each extractor's output length
        for diagnostic logging on exhaustion. (Quick 260511-b4k.)
        """
        import trafilatura
        import html2text as _html2text_mod  # alias to avoid shadowing fn arg

        lengths: dict[str, int] = {"precision": 0, "recall": 0, "html2text": 0}

        # Extractor 1: trafilatura precision (current behavior)
        md_p = trafilatura.extract(
            html,
            output_format="markdown",
            include_images=True,
            include_links=True,
            favor_precision=True,
        )
        lengths["precision"] = len(md_p) if md_p else 0
        if _passes_quality_gate(md_p):
            return md_p, {"method": "trafilatura-precision", **lengths}

        # Extractor 2: trafilatura recall (less aggressive boilerplate filter)
        md_r = trafilatura.extract(
            html,
            output_format="markdown",
            include_images=True,
            include_links=True,
            favor_recall=True,
        )
        lengths["recall"] = len(md_r) if md_r else 0
        if _passes_quality_gate(md_r):
            return md_r, {"method": "trafilatura-recall", **lengths}

        # Extractor 3: html2text (most lenient — accepts almost any HTML)
        try:
            md_h = _html2text_mod.html2text(html)
        except Exception as e:  # noqa: BLE001 — last-resort, swallow + record
            logger.warning("scraper: html2text raised %s — treating as empty", e)
            md_h = ""
        lengths["html2text"] = len(md_h) if md_h else 0
        if _passes_quality_gate(md_h):
            return md_h, {"method": "html2text", **lengths}

        return None, lengths
    ```
