Pure structural refactor with zero behavior change on the happy path.

Purpose: Today, three early-exit / exception sites between conn creation and the inner try can leak the conn — two `if not rows:` / `if not candidate_rows:` early-returns explicitly call `conn.close()` (so they're already covered), but the two await points (`await layer1_pre_filter(articles_meta)` and `await get_rag(flush=True)`) plus the schema migration / SELECT call sites have NO outer try, so any exception there leaks the conn. Wrapping the body in `with contextlib.closing(...)` makes scope-exit close mandatory and lets us delete the 3 redundant explicit `conn.close()` calls (idempotent close means surviving call sites would have been harmless, but we're cleaning up).

**Step 4 — Wrap conn creation + reindent body (the big one):**

This is the load-bearing change. The cleanest way is a one-shot Python script that:
  1. Reads the file
  2. Locates the line `    conn = sqlite3.connect(str(DB_PATH))` inside `ingest_from_db` (the only call site for that exact string in the file)
  3. Locates the end-of-function (the line `        conn.close()` at the bottom of the inner finally — which after this refactor will be at indentation level 8 spaces, will become the LAST line we touch and will be DELETED in step 5)
  4. Replaces line (2) with `    with contextlib.closing(sqlite3.connect(str(DB_PATH))) as conn:`
  5. For every line between (2)+1 and (3) inclusive, prepends 4 spaces of leading whitespace
  6. Writes the file back

Run this script via `Bash`:

```python
venv/Scripts/python.exe << 'PY'
from pathlib import Path

p = Path("batch_ingest_from_spider.py")
src = p.read_text(encoding="utf-8").splitlines(keepends=False)

# Find the conn creation line (only one of its kind in the file)
conn_idx = None
for i, line in enumerate(src):
    if line == "    conn = sqlite3.connect(str(DB_PATH))":
        assert conn_idx is None, f"duplicate match at {conn_idx} and {i}"
        conn_idx = i
assert conn_idx is not None, "could not find conn = sqlite3.connect(...) line"

# Find the trailing conn.close() at end of inner finally — sentinel:
# the last line of ingest_from_db is `        conn.close()` (8 spaces indent),
# preceded by `        logger.info("Metrics written to %s", metrics_path)`.
# We locate the line `        conn.close()` whose NEXT line begins a new top-
# level def or async def (or is the run() function start). Simpler: search
# for the unique sentinel pair.
end_idx = None
for i in range(conn_idx + 1, len(src)):
    if src[i] == "        conn.close()" and i + 1 < len(src):
        nxt = src[i + 1].lstrip()
        # Next non-empty/non-comment top-level line should be a def/async def
        # or a blank-then-def. Accept either pattern.
        j = i + 1
        while j < len(src) and src[j].strip() == "":
            j += 1
        if j < len(src) and (src[j].startswith("def ") or src[j].startswith("async def ") or src[j].startswith("# ")):
            end_idx = i
            break
assert end_idx is not None, "could not find trailing conn.close() sentinel"

# Replace conn creation with the with-statement
src[conn_idx] = "    with contextlib.closing(sqlite3.connect(str(DB_PATH))) as conn:"

# Reindent every line from conn_idx+1 through end_idx (inclusive).
# Blank lines stay blank (no trailing-whitespace pollution).
for i in range(conn_idx + 1, end_idx + 1):
    if src[i].strip() == "":
        continue  # leave blank lines blank — no leading whitespace
    src[i] = "    " + src[i]

# Write back (preserve trailing newline)
p.write_text("\n".join(src) + "\n", encoding="utf-8")
print(f"Reindented lines {conn_idx + 1}..{end_idx + 1} ({end_idx - conn_idx} lines)")
PY
```
